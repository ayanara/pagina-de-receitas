# Página de Receitas

Projeto desenvolvido como tarefa da Rocketseat, com o objetivo de praticar a criação de uma página web simples usando HTML e CSS.

## Sobre o projeto

A página apresenta uma receita de cupcake de café com chantilly, contendo:

- imagem principal da receita;
- descrição;
- tempo de preparo;
- rendimento;
- nível de dificuldade;
- lista de ingredientes;
- modo de preparo;
- rodapé com link para o GitHub.

## Tecnologias utilizadas

- HTML5
- CSS3
- Google Fonts

## Estrutura do projeto

```text
paginaReceitas/
|-- assets/
|   |-- bg-image.jpg
|   `-- main-image.png
|-- index.html
|-- style.css
`-- README.md
```

## Como visualizar

Abra o arquivo `index.html` no navegador.
